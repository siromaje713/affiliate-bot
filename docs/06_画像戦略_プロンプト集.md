# 06*画像戦略*プロンプト集

## 採用パターン：パターンA（birefnet → Flux image-to-image）

Amazon商品画像 → birefnet背景除去 → Flux背景合成 → Threads投稿添付

```
Amazon商品画像URL
  ↓ fal-ai/birefnet（背景除去・PNG出力）
  ↓ fal-ai/flux/dev/image-to-image（背景合成・strength=0.65）
  → 合成済み画像URL → Threads投稿に添付
```

実装：`image_generator.py` の `generate_product_image(product_name, image_url)`

## 設定メモ

- 出力サイズ：`square_hd`（1080×1080相当）
- num_inference_steps：28
- guidance_scale：3.5
- strength：0.65（元商品の形状を保持しつつ背景を変える）
- 人物の顔は生成しない（プロンプトに人物要素を含めない）
- FAL_KEY：環境変数から取得（Render env group `affiliate-bot-share` に設定）
- 費用目安：月600円程度（投稿数次第）
- FAL_KEY未設定時はフォールバック：元のAmazon商品画像URLをそのまま使用
